        # 获取动态详情

        https://api.bilibili.com/x/polymer/web-dynamic/v1/detail

        请求方式：GET

        是否需要登录：否

        # URL参数

        参数名 类型 必填 内容 备注
        timezone_offset num -480
        id num √ 动态id

        # Json回复

        # 根对象

        字段名 类型 内容 备注
        code num 响应码 0：成功
        4101139：4101139
        message str
        ttl num 1
        data obj 信息本体

        # data对象

        字段名 类型 内容 备注
        item obj

        # data对象 -> item对象

        参照 获取动态列表

        # 请求示例

        curl -L -X GET 'https://api.bilibili.com/x/polymer/web-dynamic/v1/detail?id=724328028268658744'
        1
